## MERN app
##### Создано для демонстрации текущих навыков в `React`,  `NodeJS` и `Typescript`.

 Использованы:
- база данных `mongodb`
- фреймворк `express`
- инструмент для настройки сред разработки. `vite`
- библиотека`react`
  - `react-router-dom`
  - `axios`
  - `react-query`
  - `react-select`
  - хуки `useCallback`, `useRef`, `useEffect`, `useContext`, `useState`, `useMemo`
  - пользовательские хуки
- `sessionStorage`
- препроцессор `scss`
- `typescript`
